# 🏙️ Dubai Property Market Analyzer

A data-driven real estate dashboard built with Python, combining 1M+ historical 
DLD transactions with live Bayut market data.

## Features
- 📊 1,000,000+ real Dubai Land Department transactions
- 🔴 Live property transaction data via Bayut API
- 🗺️ Interactive map of transactions by price
- 📈 Price trends, area comparisons, bedroom breakdowns

## Tech Stack
Python · Streamlit · Pandas · Plotly · RapidAPI

## Setup
1. Clone the repo
2. `pip install -r requirements.txt`
3. Add your RapidAPI key to `.env` as `RAPIDAPI_KEY=your_key`
4. Download DLD transactions CSV from Dubai Pulse and rename to `transactions.csv`
5. `streamlit run app.py`
